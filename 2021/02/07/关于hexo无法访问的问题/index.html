关于hexo无法访问的问题

发表于 2021-02-07

大概一周以前开始搭建hexo，听网友广泛反应自建初期可能会遇到很多奇葩的bug，不信，跟着一些讲得轻巧的教程过一遍才体会深刻。
其中可能遇到的问题可能是文件夹套文件夹，D:/hexo/hexo/报错，此时需要删掉内嵌的目录。
亲测SSH密钥设置后无法识别，还是弹出GitHub登录框输入登录名与密码登录为好。
另外如：Branch ‘master’ set up to track remote branch ‘master’ from ‘git@github.com:7msunshine/7msunshine.github.io.git’.
INFO Deploy done: git所示，并不是一种报错，虽然可能因为浏览器缓存的问题，无法及时更新，刷新缓存即可。
本地可以预览，无法同步时使用hexo deploy -g同步上传即可。
